## ① 前期准备（开拍前必做）

### 环境检查

```
□ 关闭所有无关窗口（微信、浏览器其他标签页、通知弹窗）
□ 浏览器打开隐身窗口（避免历史记录/书签栏干扰）
□ 调整浏览器窗口为 16:9 比例（推荐 1920×1080 或 1280×720）
□ 确认麦克风正常，环境安静
□ 准备一个记事本在旁边写关键词提示
```

```bash
cd D:\作业\全球数字创业\任务pt.2\mvp

# 看到 "Running on http://127.0.0.1:5000" 即可
```

## ④ 完整生成演示（核心，2分30秒）

### 动作流程（精确到每次点击）

**第1步：选题材（5秒）**
- 点击 🏫"校园奇幻"卡片（它已经是默认选中，演示点击其他再点回来）
- 看到卡片边框变成实线粉色，背景高亮

> 口播："先选题材。这里选校园奇幻。"

**第2步：输入创意（10秒）**
- 点击创意文本框
- 输入："一只会说话的猫在魔法学院里揭开了地下王国的秘密，却发现自己是千年前被封印的守护灵"
- 或者点击快捷标签 "🐱 会说话的猫" 自动填入

> 口播："然后输入你的创意想法。可以是一句话，也可以点下面的快捷标签。这里我写的是一个校园奇幻故事——会说话的猫、魔法学院、地下王国的秘密。"

**第3步：配置参数（15秒）**
- 点击集数的"+"按钮调到 2 集
> 口播："集数调成两集，方便演示。"
- 点击画风下拉框，选中 "🎌 日系二次元 (Anime)"
> 口播："画风选日系二次元——这是AI漫剧创作者最常用的风格之一。"

**第4步：选择平台（15秒）**
- 点击 "🖼️ 图像生成平台" 下拉按钮
- 展示下拉菜单滑出动画
- 选择 "Midjourney"
- 标签更新为 "Midjourney"，菜单自动收起
> 口播："图像平台选Midjourney。注意这里是单选——我们为每个平台专门优化了提示词格式。"
- 点击 "🎬 视频生成平台" 下拉按钮
- 选择 "Pika / PixVerse"
- 标签更新，菜单自动收起
> 口播："视频平台选Pika。不同平台的提示词格式完全不同——Pika需要显式的运镜方向和角色动作描述。"

**第5步：点击生成（等待期间讲解，60秒）**
- 鼠标移到 "✨ 生成剧本" 按钮上
- **点击**

> 口播："点击生成。现在是见证时刻——"

**等待阶段的口播（边看动画边讲，不要沉默）：**

- 进度条开始流动时（0-10秒）：

## ⑥ 结果浏览（1分20秒）

### 逐Tab展示（每个Tab约10-15秒）

**Tab 1：① 剧本大纲（15秒）**
- 点击 "① 剧本大纲" Tab
- 滚动展示世界观→故事梗概→分集大纲
- 鼠标指向分集大纲末尾的 "[HOOK]" 标记

> 口播："第一个模块是剧本大纲。包括世界观设定、故事梗概，以及逐集的分集大纲。注意每集结尾都有一个[HOOK]标记——这是我们prompt里要求的悬念钩子，适配短视频平台'每集结尾留悬念'的创作规律。"

**Tab 2：② 角色设定（15秒）**
- 点击 "② 角色设定" Tab
- 滚动展示角色外貌、服装、标志动作
- 鼠标指向"参考图提示词"
> 口播："第二个模块是角色设定。每个角色有详细的外貌描述——这种描述本身就是AI生图prompt的风格。最下面还有一个参考图提示词，可以直接复制到Midjourney生成角色立绘。"

**Tab 3：③ 场景描述（10秒）**
- 点击 "③ 场景描述" Tab
> 口播："场景描述。每个场景有时间、地点、画面描述和氛围，同样附带参考图提示词。"

**Tab 4：④ 分集台词（10秒）**
- 点击 "④ 分集台词" Tab
- 展示对话格式（角色：台词 + 动作描写）
- 指向末尾的【本集钩子】
> 口播："分集台词。格式是场景标题加角色对话，动作描写用括号标注。每集控制在300到500字，适配1到1.5分钟的短视频时长。"

**Tab 5：⑤ 分镜脚本（10秒）**
- 点击 "⑤ 分镜脚本" Tab
- 展示分镜表格（镜号/景别/画面/运镜/时长/转场）
> 口播："分镜脚本。表格格式，每集5到8个镜头。镜号、景别、画面描述、运镜方式、时长、转场方式——一应俱全。"

**Tab 6：⑥ AI提示词（⭐重点，20秒）**
- 点击 "⑥ AI提示词" Tab
- **这是最关键的一页**
- 鼠标指向英文prompt
- 指向 "SELECTED IMAGE PLATFORM: Midjourney"
- 指向 "SELECTED VIDEO PLATFORM: Pika / PixVerse"

> 口播："这是最关键的一个模块——AI生成提示词。注意，因为我们在配置里选了Midjourney和Pika，所以这个模块里的所有提示词都是针对这两个平台专门优化的格式。Midjourney的加了niji 6动漫模型参数，Pika的加了显式的运动方向和强度描述。这些提示词可以直接复制粘贴到对应工具里使用——不需要任何手动调整。这就是我们产品最核心的价值：AI原生格式，即拿即用。"

## 🎬 拍摄技巧

| 技巧 | 说明 |
|------|------|
| **鼠标移动要慢** | 每个指向停留1-2秒，让观众看清 |
| **不要滚动太快** | 结果展示时慢慢滚，每个模块停一下 |
| **口播要自然** | 不要念稿感，像在给朋友演示 |
| **备选方案** | 如果生成超过90秒还没完，切换到预先生成好的结果 |
| **高亮点击** | OBS可以开启"显示鼠标点击"效果 |
| **画面缩放** | 关键文字（如AI提示词）可以用OBS的画面缩放功能放大 |
