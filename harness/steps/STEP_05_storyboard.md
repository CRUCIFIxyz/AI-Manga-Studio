# STEP 05: 分镜脚本 (Storyboard)

> 流水线第5步。根据角色外貌、场景描述和台词，生成逐镜分镜表。本步骤是**一致性关键节点**——必须引用前面模块的角色名和场景名。

---

## Input Variables

| 变量 | 来源 | 类型 | 必填 |
|------|------|------|:--:|
| `{genre_cn}` | 用户选择 | 题材中文名 | ✅ |
| `{episode_count}` | 用户配置 | 集数 | ✅ |
| `{character_appearances}` | STEP 02 输出 | 角色外貌Map | ✅ |
| `{scene_descriptions}` | STEP 03 输出 | 场景描述Map | ✅ |
| `{dialogue_summary}` | STEP 04 输出 | 台词摘要 | ✅ |

---

## System Prompt Template

```
你是一位分镜师，专精于「{genre_cn}」题材。

根据以下素材，写出逐镜分镜表。

【角色外貌（分镜中必须使用以下名称和描述）】
{character_appearances}

【场景描述（分镜中必须使用以下场景名）】
{scene_descriptions}

【台词摘要】
{dialogue_summary}

【要求】
- 共 {episode_count} 集，每集5-8个镜头
- 表格格式：镜号 | 景别 | 画面描述 | 运镜 | 时长 | 转场
- 画面描述中必须引用上述角色名和场景名——禁止改名
- 以下格式输出，不要输出其他内容
```

---

## Output Format

```
### 第1集分镜

| 镜号 | 景别 | 画面描述 | 运镜 | 时长 | 转场 |
|------|------|---------|------|------|------|
| S101 | 全景/中景/近景/特写 | {角色名}在{场景名}中做什么。光线、构图描述。 | 推/拉/摇/移/跟/静止 | Xs | 切/淡入/淡出/叠化 |
| S102 | ... | ... | ... | Xs | ... |
（继续 S103 ~ S108）

### 第2集分镜
...
```

---

## Validation Rules

| 规则ID | 检查项 | 条件 | 失败动作 |
|--------|--------|------|---------|
| V05-01 | 集数与 {episode_count} 一致 | count == {episode_count} | RETRY |
| V05-02 | 每集 ≥ 5 个镜头 | 逐集检查 | RETRY |
| V05-03 | 镜头描述中至少引用1个角色名 | 逐镜检查 | RETRY |
| V05-04 | 景别字段在 [全景/中景/近景/特写] 中 | 逐镜检查 | WARN |
| V05-05 | 运镜字段在 [推/拉/摇/移/跟/静止] 中 | 逐镜检查 | WARN |
| V05-06 | 使用了 STEP 02 中的正确角色名 | 新角色名=0 | RETRY |
