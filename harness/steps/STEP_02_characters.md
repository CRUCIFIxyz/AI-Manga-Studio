# STEP 02: 角色设定 (Characters)

> 流水线第2步。根据大纲摘要生成主要角色的详细设定，包含可直接用于AI生图的描述。

---

## Input Variables

| 变量 | 来源 | 类型 | 必填 |
|------|------|------|:--:|
| `{genre_cn}` | 用户选择 | 题材中文名 | ✅ |
| `{art_desc}` | 用户画风选择 | 画风英文描述 | ✅ |
| `{outline_summary}` | STEP 01 输出 | 大纲摘要 (200字) | ✅ |

---

## System Prompt Template

```
你是一位角色设计师，专精于「{genre_cn}」题材。

根据以下故事大纲，设计主要角色。

【故事大纲摘要】
{outline_summary}

【画风要求】
{art_desc}

【要求】
- 至少设计2个角色（主角+重要配角）
- 角色外貌描述必须可直接用于AI生图（包含发色、瞳色、体型、服装颜色）
- 以下格式输出，不要输出其他内容
```

---

## Output Format

```
### 角色一：[角色名]
- **姓名**：（中文名）
- **年龄**：（数字）
- **外貌**：（4-5句——发色/发型/瞳色/脸型/身高体型/标志特征。必须包含至少3个可搜索关键词）
- **服装**：（2-3句——颜色/材质/标志性配饰）
- **性格**：（2-3个关键词 + 1句说明）
- **标志动作**：（1个招牌动作或习惯）
- **参考图提示词**：（1句英文——可直接用于即梦/Midjourney生成角色立绘）

### 角色二：[角色名]
（同上格式）
```

---

## Validation Rules

| 规则ID | 检查项 | 条件 | 失败动作 |
|--------|--------|------|---------|
| V02-01 | 角色数 ≥ 2 | count ≥ 2 | RETRY |
| V02-02 | 每个角色的"外貌"字段 ≥ 80 字符 | 逐角色检查 | RETRY |
| V02-03 | 每个角色含至少3个外貌关键词（发色/瞳色/服色） | 正则提取 | WARN |
| V02-04 | 每个角色含"参考图提示词" | 字段存在 | RETRY |
| V02-05 | 角色名不含英文或数字 | 正则: `^[\u4e00-\u9fff·]{2,6}$` | WARN |
