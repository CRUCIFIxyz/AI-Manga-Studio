# STEP 03: 场景描述 (Scenes)

> 流水线第3步。根据故事背景和角色信息，生成关键场景的详细视觉描述。场景名将被STEP 04/05/06引用。

---

## Input Variables

| 变量 | 来源 | 类型 | 必填 |
|------|------|------|:--:|
| `{genre_cn}` | 用户选择 | 题材中文名 | ✅ |
| `{art_desc}` | 用户画风选择 | 画风英文描述 | ✅ |
| `{character_names}` | STEP 02 输出 | 角色名列表 | ✅ |
| `{outline_full}` | STEP 01 输出 | 大纲全文 | ✅ |

---

## System Prompt Template

```
你是一位场景设计师，专精于「{genre_cn}」题材。

根据以下故事大纲和角色信息，设计故事中的关键场景。场景名将作为后续所有模块的"位置锚点"——不可改名。

【故事大纲】
{outline_full}

【角色列表（这些角色将出现在以下场景中）】
{character_names}

【画风要求】
{art_desc}

【硬性要求】
- 列出6-8个关键场景
- 每个场景的"画面描述"必须是可直接用于AI生图的视觉描述——光线、色彩、材质、空间、天气，至少5句话，≥100字
- 每个场景必须附一段完整英文参考图提示词
- 场景名应简洁——2-5个字（如"魔法教室""星空操场"）
- 严格按以下格式输出
```

---

## Output Format

```
### 场景一：[场景名]
- **时间**：（晨/午/黄昏/夜）
- **地点**：（具体位置——不只是"教室"，而是"魔法学院三楼走廊尽头的废弃教室"）
- **画面描述**：（≥5句话。必须含光线来源+方向、主色调、关键材质、空间深度、天气/氛围元素。≥100字）
- **氛围**：（1-2个词——神秘/温暖/压抑/空灵）
- **参考图提示词**：（≥1句完整英文，可直接用于即梦/Flux生成场景图）

### 场景二：[场景名]
（同上格式）
```

---

## Validation Rules

| 规则ID | 检查项 | 条件 | 失败动作 |
|--------|--------|------|---------|
| V03-01 | 场景数 ≥ 6 | count ≥ 6 | RETRY |
| V03-02 | 每个场景的"画面描述" ≥ 100字符 | 逐场景检查 | RETRY |
| V03-03 | 每个场景含"参考图提示词"且 ≥ 30字符 | 字段存在+长度 | RETRY |
| V03-04 | 每个场景指定了"时间"和"地点" | 字段非空 | WARN |
| V03-05 | 场景名不含超过10字的冗长名称 | 逐场景检查 | WARN |
