# STEP 03: 场景描述 (Scenes)

> 流水线第3步。根据故事背景和角色信息，生成关键场景的详细视觉描述。

---

## Input Variables

| 变量 | 来源 | 类型 | 必填 |
|------|------|------|:--:|
| `{genre_cn}` | 用户选择 | 题材中文名 | ✅ |
| `{art_desc}` | 用户画风选择 | 画风英文描述 | ✅ |
| `{character_names}` | STEP 02 输出 | 角色名列表 | ✅ |

---

## System Prompt Template

```
你是一位场景设计师，专精于「{genre_cn}」题材。

根据以下角色信息，设计故事中的关键场景。

【角色列表】
{character_names}

【画风要求】
{art_desc}

【要求】
- 列出6-8个关键场景
- 每个场景的画面描述必须可直接用于AI生图（包含光线、色彩、空间、天气）
- 每个场景附一段英文参考图提示词
- 以下格式输出，不要输出其他内容
```

---

## Output Format

```
### 场景一：[场景名]
- **时间**：（晨/午/黄昏/夜）
- **地点**：（具体位置）
- **画面描述**：（4-5句——光线、色彩、材质、空间布局、天气。必须包含至少3个可搜索关键词）
- **氛围**：（1-2个词）
- **参考图提示词**：（1句英文——可直接用于即梦/Flux生成场景图）

### 场景二：[场景名]
（同上格式）
...
```

---

## Validation Rules

| 规则ID | 检查项 | 条件 | 失败动作 |
|--------|--------|------|---------|
| V03-01 | 场景数 ≥ 5 | count ≥ 5 | RETRY |
| V03-02 | 每个场景的"画面描述"字段 ≥ 80 字符 | 逐场景检查 | RETRY |
| V03-03 | 每个场景含"参考图提示词" | 字段存在 | RETRY |
| V03-04 | 每个场景指定了"时间" | 字段非空 | WARN |

---
